# Noisy Drone RF v2 Result Card

This folder is the committed home for Noisy Drone RF Signal Classification v2 evaluation evidence generated by `notebooks/44_evaluation_noisy_drone_rf_v2.ipynb`. The matching training notebook is `notebooks/33_cnn_bilstm_noisy_drone_rf_v2.ipynb`.

## Result Snapshot

| Field | Value |
|---|---|
| Dataset | Noisy Drone RF Signal Classification v2 |
| Data caveat | Public preprocessed/noise-augmented I/Q data; not over-the-air Navy field data |
| Model | CNN + BiLSTM over fixed-length I/Q windows |
| Model path | `models/noisy_drone_rf_v2/noisy_drone_rf_v2_cnn_bilstm_final.keras` |
| Split method | Stratified 80/20 held-out test split |
| Seed | 1961 |
| Classes | 7 |
| Eval support | 203 samples: 29 held-out windows per class |
| Balanced accuracy | 0.9803 |
| Macro F1 | 0.98 |
| Weighted F1 | 0.98 |

## Interpretation Notes

- "1 eval windows" in the notebook output means one fixed-length model input window per source `.pt` sample after trimming to the common sequence length. It does not mean only one total evaluation example.
- The visible weakness in the local run is small: Noise precision is 0.88, so a few drone/controller samples are predicted as noise, and DJI recall is 0.93.
- The local SNR accuracy curve appears strong from low to high SNR.
