# Dataset Card: RML2016.10A

## Source

RadioML 2016.10A modulation dataset, loaded by this repo from `data/RML2016/RML2016.10a_dict.pkl` or a configured local path.

## Class List

`8PSK`, `AM-DSB`, `AM-SSB`, `BPSK`, `CPFSK`, `GFSK`, `PAM4`, `QAM16`, `QAM64`, `QPSK`, `WBFM`.

## SNR Range

`-20 dB` to `+18 dB` in 2 dB increments.

## Sample Count

Common RML2016.10A layout: 220,000 examples total across 11 classes and 20 SNR bins. The repo comparison artifact reports 44,000 all-SNR test examples for the current split.

## Preprocessing Assumptions

- Dataset is a pickle dictionary keyed by `(modulation, snr)`.
- Raw samples are 128-timestep I/Q sequences.
- Some repo models append SNR as a third channel, yielding `(128, 3)`.

## License / Usage Limitations

Use according to the original RadioML dataset terms. Do not assume this repository grants redistribution rights for the raw dataset.

## Leakage Risks

- Split at the sample level within each `(modulation, snr)` group.
- Keep SNR-slice metrics clearly labeled.
- Do not report high-SNR-only performance as full-distribution performance.

## Recommended Evaluation Split

Use a stratified train/test split and report:

- all-SNR test metrics
- high-SNR slice metrics
- sample count for every filtered slice
